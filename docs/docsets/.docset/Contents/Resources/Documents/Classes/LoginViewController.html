LoginViewController

class LoginViewController : UIViewController, GIDSignInUIDelegate

The first view users see when they aren’t signed in. It is used to sign in with both an email/password combination and Google sign-in.

The Firebase documentation by Google was used as guide for everything related to signing in. SOURCE: https://firebase.google.com/docs/auth/ios/custom-auth SOURCE: https://firebase.google.com/docs/auth/ios/google-signin
